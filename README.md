📚 **Bookstore MERN Application**


🚀  Features

User Features 🛒 Add/remove books from the cart. 💳 Checkout and place orders (cash on delivery). 📄 View order history and details.

Admin Features 📚 Add, update, and delete books securely. 📊 Admin dashboard for managing inventory. 📦 Robust inventory tracking.

General Features 🔒 Secure user authentication using JWT (JSON Web Tokens). 🌐 Dynamic and responsive UI designed with Tailwind CSS. 🔄 State management and API handling with Redux Toolkit and RTK Query.

🛠️ Tech Stack

Frontend React.js Tailwind CSS Redux Toolkit (with RTK Query for API management)

Backend Node.js Express.js JWT for secure authentication Database MongoDB with Mongoose
